Run example

Step by step run example with CS.

This page will show step by step procedure of the differential cross section plots production as an example. To produce all dimuon or secondary vertices distributions one should make similar steps as described in the "General procedure" part below, but with relevant scripts which one can find at the autoscripts folder.

General procedure

1. Work with miniNTuples, use cuts, all GutCode procedures, and get nextTominiNTuples, which will be stored inside $GUTANADIR/data and $GUTANADIR/mc.

cd $GUTANADIR/autoscripts/
sh run_all_years.sh

2. Combined script which creates
"mini_HeraII_0307p_Data_MC_xsec2_Finalversion_dbothistos_dimuon" folder at the
$GUTANADIR, extract all needed histograms from data and mc folders and crete 4 folders :

"mini_HeraII_0*_Data_MC_Dimuon_Finalversion_XsectionTotal2_*_dbothistos".
In each folder root files with histograms are stored. Then from 4 folders those files are moved into "mini_HeraII_0307p_Data_MC_xsec2_Finalversion_dbothistos_dimuon".
In this folder root files are merged. Next part - extraction txt files from the merged root files.

From the text files there is the possbility to produce cross section plots with using the script "Crosssection_Extraction.C".

cd $GUTANADIR/autoscripts/
sh finilazer_xsec2.sh

More detailed description

"run_all_years.sh" Use the $GUTANADIR/analyse script, and 4 cards from $GUTCODEDIR/analysis/cards/templates/

dbot_general_gmuon_dimuon_analyse_12_HeraII_Switch_0304p.cards
dbot_general_gmuon_dimuon_analyse_12_HeraII_Switch_05e.cards
dbot_general_gmuon_dimuon_analyse_12_HeraII_Switch_06e.cards
dbot_general_gmuon_dimuon_analyse_12_HeraII_Switch_0607p.cards

And "mini_HeraII_0*_Data_MC_Dimuon_Finalversion" will be core name for the output root files. Cards contain set of all cuts. $GUTCODEDIR/libraries/GutLib/src/ contain all C++ classes to run and work with miniNTuples.

"finilazer_xsec2.sh" is a script which contain few steering scripts:

$GUTANADIR/Extraction_Scripts/run_all_xsection2.sh
$GUTANADIR/autoscripts/move_all_xsec2.sh
$GUTANADIR/mini_HeraII_0307p_Data_MC_xsec2_Finalversion_dbothistos_dimuon/mergefiles.sh
$GUTANADIR/mini_HeraII_0307p_Data_MC_xsec2_Finalversion_dbothistos_dimuon/Cross_Sectionscript.sh

"run_all_xsection2.sh" is a script which contain few steering scripts:

...
sh getdanny_XsectionTotal2_05e.sh mini_HeraII_05e_Data_MC_Dimuon_Finalversion
sh getdanny_XsectionTotal2_06e.sh mini_HeraII_06e_Data_MC_Dimuon_Finalversion
...

creates the folder:
mini_HeraII_0304p_Data_MC_Dimuon_Finalversion_XsectionTotal2_0304p_dbothistos, with 29 root files. There will be 3 other folders for the rest 3 year set.

"move_all_xsec2.sh" move all root files from such folder:

mini_HeraII_0304p_Data_MC_Dimuon_Finalversion_XsectionTotal2_0304p_dbothistos

into the combined folder

mini_HeraII_0307p_Data_MC_xsec2_Finalversion_dbothistos_dimuon

plus copy needed .sh and .C files there.

"mergefiles.sh" merge root files from the 4 time periods inside the

mini_HeraII_0307p_Data_MC_xsec2_Finalversion_dbothistos_dimuon

"Cross_Sectionscript.sh" new set of text files, from the root files inside the folder showed above. Creates text files : h*, uMC_*, recMC_*, true_* , uData_*. (h* - 120 files, the rest - 120 files as well).

"Plot_Cross_Sections.C" Produce CS differential distributions.
