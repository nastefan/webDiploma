What is new

The program has possibilities:

-- extract and loop over all gen particles;
-- identify gen particles by id
-- extract complete set of parameters for the gen particle (p, Eta, Phi, so on)
-- follow complete decay chain (find the daughter/mother of the gen particle and decay status)
-- this program I've checked on 2011 VM and it works well with 2011 MC samples.
-- since this is done for 2011 , release is CMSSW_5_3_32.
-- show complete true decay information in the table/chain format.
