HelixTrans Class

Simple information what function of this class does

General Information

This is a new class which has been written to make easy usage of VXLite and CMS vertex fitters. It also can transform track helix parameters from CMS format into ZEUS format. This class basically consist of two files:

Demo/DemoAnalyzer/src/HelixTransClass.cxx Demo/DemoAnalyzer/include/HelixTransClass.h
One can study HelixTransClass.cxx file to understand what this class can do, in details.

Class function description

Let say you create class object as HelixTrans ZeusVertexFit(false);

Set track Builder:

ZeusVertexFit.SetBuilder(theB);

Set magnetic field:

ZeusVertexFit.SetMagneticField(BField);

Set Beam Spot Information:

ZeusVertexFit.SetBeamSpotInformation(beamSpotHandle_I);

It is used for "GetTracksNotFromPVs" function. Can use on input user defined track collection:

ZeusVertexFit.SetGeneralTrackCollection(AllgenralTracks);

Set tracks from PV only here:

vector tracksFromPVcheck = ZeusVertexFit.GetTracksFromPV(ite);
ZeusVertexFit.SetTrackCollectionForAlgorithm(tracksFromPVcheck);
ZeusVertexFit.SetTrackCollectionToFit(mytracks);

Use the CMS or VXLite fitters:

vector VtxParametersZEUScheck;
VtxParametersZEUScheck = ZeusVertexFit.SimpleFitWithFitterCMS(mytracks, ite, false);
VtxParametersZEUScheck = ZeusVertexFit.SimpleFitWithVxlite(mytracks, ite, false);
VtxParametersZEUScheck contains complete set of parameters of the fitted vertex. Detailed description of such paramters can be founded in HelixTransClass.cxx, in function SimpleFitWithFitterCMS.

Look! Example of such class usage is present in the DemoAnalyzer.cc file, starts at the 2140 line.
